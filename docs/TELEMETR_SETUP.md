# Бесплатный старт через Telemetrio

## Что даёт бесплатный тариф

На странице тарифов Telemetrio для Free указаны одно отслеживание, неограниченное количество слов и каналов внутри отслеживания, лимит 100 упоминаний и уведомления в Telegram.

Этого достаточно, чтобы без затрат проверить спрос и откалибровать фильтры ParcerTG.

## Настройка

1. Зарегистрируйтесь в Telemetrio.
2. Откройте инструмент отслеживания событий/упоминаний.
3. Создайте одно отслеживание.
4. Добавьте строки из `config/telemetr_keywords.txt` в ключевые слова и фразы.
5. Добавьте строки из `config/telemetr_minus_words.txt` в минус-слова.
6. Подключите Telegram-бот уведомлений Telemetrio.
7. Каждое потенциально подходящее уведомление пересылайте в ParcerTG.

Telemetrio самостоятельно выполняет широкий поиск, а ParcerTG отвечает за второй слой:

- оценку намерения нанять;
- проверку совпадения с твоим стеком;
- отсечение резюме и рекламы;
- дедупликацию;
- учёт результата.

## Как работать быстрее

Пересылка в ParcerTG не требует команды. Просто нажмите «Переслать» в уведомлении Telemetrio и выберите своего бота.

Для текста, полученного вне Telemetrio:

```text
/lead Нужен Telegram-бот для обработки заявок и интеграции с CRM. Бюджет 80 000 ₽.
```

Для проверки качества фильтра без сохранения:

```text
/check Нужен Python-разработчик для FastAPI, начать на этой неделе.
```

## Когда переходить на платный источник

Переход к TGStat Callback имеет смысл после одного из событий:

- бесплатные 100 упоминаний закончились слишком быстро;
- ручная пересылка стала отнимать заметное время;
- найден хотя бы один оплаченный заказ;
- подтверждена приемлемая доля релевантных сообщений.
