# План запуска с минимальными расходами

## Этап 1 — 0 ₽

- `SOURCE_PROVIDERS=manual`;
- одно бесплатное отслеживание Telemetrio;
- первые 100 упоминаний;
- пересылка уведомлений в ParcerTG;
- сбор статистики по команде `/stats`.

Цель: получить не меньше 30–50 обработанных сообщений и понять долю настоящих заказчиков.

## Этап 2 — недорогой ручной мониторинг

Если Telemetrio подтвердил спрос, можно протестировать одну персональную тему TGStat. Уведомления также пересылаются в ParcerTG, поэтому код менять не нужно.

Цель: сравнить охват и качество двух сервисов в течение одного месяца.

## Этап 3 — полная автоматизация

После первого оплаченного заказа или устойчивого потока включить `tgstat` и приобрести минимальный API Callback, достаточный для одной keyword-подписки.

```env
SOURCE_PROVIDERS=manual,tgstat
```

Цель: убрать ручную пересылку, сохранив один составной запрос и минимальную квоту объектов.

## Этап 4 — дополнительный охват

Если удастся получить собственные `api_id/api_hash`, добавить `telethon` и мониторить каталог из 100 источников параллельно:

```env
SOURCE_PROVIDERS=manual,tgstat,telethon
```

TGStat и Telethon не создают двойных карточек: общий процессор объединяет одинаковые сообщения.
